Leo Latvalas's weather dashboard

A weather dashboard website I made as my last course project for my Univeristy.
It shows temperature, rain , and wind data.

Live Demo:

http://172.16.5.34/weather_dashboard/

What it does
-Shows weather information for Tampere,Finland.
-Lets you picd dates to see the past
-Has dark mode feature for those who are sensitive to bright lights
-Makes simple cahrts and shows basic mathematical values e.g. average, min, max etc.

Files
-index.html -> Webpage structure
-style.css  -> adds style to the website
-script.js  -> The code that does everything

How to use
-Go to the link above
-Click buttons to switch between temp/rain/wind
-Pick dates if you want
-Click the circle button for dark mode

Tools used
-Bootstrap -> for less CSS coding
-Chart.js -> for graphs
-Font awesome -> icons
-Open-Meteo API -> weather data

Notes
-Only shows Tampere data
-Uses the last 20 readings in the table
-The VM is running 24/7 so it should always be up.
