Tooltips and Popovers

Learn Bootstrap tooltips (hover) and popovers (click)

Tooltips (Hover to see)

Tooltips appear when you hover over an element. They show small help text.

ðŸ”— Link with Tooltip
How it works:

Add these attributes to any element:

data-bs-toggle="tooltip" title="Your text here"

Plus JavaScript initialization (see bottom of page).

Popovers (Click to see)

Popovers appear when you click an element. They can contain more content than tooltips.

Popover features:
  • Has a title and content
  • Can contain HTML
  • Can be positioned (top, right, bottom, left)
  • Use data-bs-trigger="focus" to close when clicking elsewhere

Cool note:

Tooltips and popovers require JavaScript initialization. Just adding the HTML attributes is not enough!

Check the JavaScript code at the bottom of this page to see how to initialize them.

<!-- Tooltip Example -->
<button data-bs-toggle="tooltip" 
        title="Help text here">
    Hover Me
</button>

<!-- Popover Example -->
<button data-bs-toggle="popover"
        data-bs-title="Title"
        data-bs-content="Content">
    Click Me
</button>
